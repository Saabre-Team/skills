# Brakson Skills v1

Plugin officiel Brakson — regroupe l'ensemble des skills Claude de l'entreprise en un seul fichier à installer.

## Installation

Double-cliquer sur `brakson-skills-v1.plugin` dans Claude Cowork. Tous les skills sont immédiatement disponibles.
